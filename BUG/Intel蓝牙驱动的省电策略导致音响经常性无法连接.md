# 🐛 Bug Report: 英特尔蓝牙驱动省电策略（Intel Bluetooth driver power-saving strategy ）导致蓝牙设备连接困难问题

## 📝 问题描述


windows连接小米蓝牙音箱经常性失败、每次只能电脑重启才能顺利连接小米音箱

## 🔗 参考资料

Gemini3 Thinking:

这是一个非常经典且令人头疼的 Windows 蓝牙连接问题。既然音箱能连手机且能被发现，说明**音箱本身没有问题**。

问题核心在于 **Windows 的蓝牙电源管理策略** 或 **蓝牙服务挂起**。简单来说，就是电脑为了省电把蓝牙模块“半休眠”了，或者蓝牙驱动在唤醒时“起床气”很重，导致无法正常握手，只有重启电脑才能强制重置这种状态。

不需要每次都重启电脑。请按顺序尝试以下解决方案，由易到难，通常**方案一**就能解决大部分人的问题。

---

### 方案一：关闭蓝牙的“省电模式” (最可能的根源)

Windows 默认允许系统为了省电而关闭蓝牙适配器，这经常导致再次连接时唤醒失败。

1. **右键点击** 屏幕左下角的“开始”按钮 (Windows徽标)。
    
2. 选择 **【设备管理器】 (Device Manager)**。
    
3. 找到 **【蓝牙】** 选项并展开。
    
4. 找到你的电脑自带的蓝牙模块（通常叫 `Intel Wireless Bluetooth`、`Realtek Bluetooth Adapter` 或 `Qualcomm` 等，**不要**选那个“Xiaomi Speaker”）。
    
5. **右键点击** 该模块，选择 **【属性】**。
    
6. 点击顶部的 **【电源管理】** 选项卡（如果没有此选项卡，请跳至方案二）。
    
7. **取消勾选** “允许计算机关闭此设备以节约电源”。
    
8. 点击 **确定**。
    

> **注意：** 修改后，建议重启一次电脑让设置生效。以后应该就不会频繁断连了。
