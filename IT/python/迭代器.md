## 迭代器
### 概念：不依赖索引的迭代取值方式
---
## 可迭代对象
```python
-可迭代对象：可以转化为迭代器的对象就是可迭代对象
-只要内置有__inter__()方法的就可以称为可迭代对象
```

```python
l = [1, 2, 4, 3]  
l.__iter__()  
  
t = '张大仙'  
t.__iter__()  

l = [1, 2, 4, 3]  
res = l.__iter__()  
print(res.__next__())  
print(res.__next__())  
  
# 1  
# 2
  
f = (1, 2, 3)  
res = f.__iter__()  
print(res)

# res 就是迭代器对象、内置有__next__()方法
# <tuple_iterator object at 0x000001702D222F50>
...
```
```python
# while版本的迭代取值
l = [2, 43 ,1 ,5]  
res = l.__iter__()  
while True:  
    try:  
        print(res.__next__())  
    except StopIteration:  
        break

2
43
1
5
```
**注意**:
	迭代器对象被创建后、只能循环调用一次、调用一个数据就少一个数据、如果想要再次重复调用、只能再创建一个迭代器对象。

## 迭代器对象
内置有__next__()方法、也内置了__iter__方法（主要是为了和 for 迭代循环保持一致）
迭代器对象的__iter__()对象还是**迭代器本身**
```python
l = [0, 2, 4, 1, 5]  
res = l.__iter__()  
res1 = res.__iter__()  
print(res is res1)  
# True
```

## 生成器
---
要求：每次只产生一个值、调用一次产生一次。（有利于节省电脑资源）
## yield
#yield

---
```python
def task():  
    print("第一次执行")  
    yield 1  
    print("第二次执行")  
    yield 2  
    print("第三次执行")  
    yield 3  
    print("第四次执行")  
    yield 4  
  
res = task()  
# print(res)  
y = res.__next__()  
x = next(res)
print(x, y)

# <generator object task at 0x000002051887E0A0>
发生器、生成器（相当于迭代器）
```
